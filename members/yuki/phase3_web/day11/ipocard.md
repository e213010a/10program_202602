#### 2. コードリーディング

以下のファイルを読み、IPOカードに書き起こしてください：

**対象ファイル：**

- `src/server.js` - サーバーのメイン処理
- `src/public/index.html` - クライアント側のUI

#### 3. IPOカード作成

Base Appの各機能について、IPOカードを作成：

**機能1: アイテムの追加**
| 項目 | 内容 |
|------|------|
| **Input** | Baseappページの起動 |
| **Process** | serverからitem全件取得 |
| **Output** | 一覧をJSON形式で返す |

**機能2: アイテムの一覧表示**
| 項目 | 内容 |
|------|------|
| **Input** | ブラウザからitemのtitleを受け取る |
| **Process** | サーバーがDBに新しいitemを保存 |
| **Output** | 作成したitemをJSON形式で返す |

**機能3: アイテムの削除**
| 項目 | 内容 |
|------|------|
| **Input** | /api/items/:idから選択されたidを受け取る |
| **Process** | DBから該当idのitemを削除 |
| **Output** | '削除しました'を出力 |

#### 4. 構造図の作成

```
┌─────────────┐                        ┌─────────────┐                        ┌─────────────┐
│   Client    │                        │   Server    │                        │  Database   │
│             │  リクエスト(GET/POST)    │             │  SQL(検索・保存・削除)    │             │
│  index.html │ ──────────────────►    │  server.js  │ ──────────────────►    │  items.db   │
│             │ ◄──────────────────    │             │ ◄──────────────────    │             │
└─────────────┘   JSON(データ一覧)      └─────────────┘   JSONデータ(結果)         └─────────────┘

各矢印に「何が送られているか」を書き込む。
```
